
  #   AWS 3-Tier Architecture using Terraform
Overview

This project demonstrates deployment of a highly available and scalable 3-tier web application architecture on AWS using Terraform.

Technologies Used
AWS
Terraform
EC2
VPC
ALB
RDS MySQL
Auto Scaling
Security Groups
CloudWatch
